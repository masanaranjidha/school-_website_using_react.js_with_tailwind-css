# School Website (React + Tailwind CSS)

## How to run
1. Unzip the folder.
2. In the project directory, run:
   ```
   npm install
   npm start
   ```
3. The site will open at http://localhost:3000

## Notes
- This project uses Create React App + Tailwind CSS (PostCSS).
- If Tailwind classes don't work right away, ensure `tailwindcss`, `postcss`, and `autoprefixer` are installed (they are listed in package.json).
